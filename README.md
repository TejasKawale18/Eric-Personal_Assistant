# Eric - Your Personal Assistant

Eric is an intelligent personal assistant designed to streamline tasks, provide insightful answers, and enhance productivity. Whether you need reminders, quick research, or everyday assistance, Eric is always ready to help.

## Features
- **Task Management**: Set reminders, to-do lists, and event scheduling.
- **Smart Responses**: Get accurate answers to queries from various domains.
- **Voice & Text Interaction**: Communicate using voice commands or text input.
- **Customizable & Adaptive**: Learns from user preferences to offer a tailored experience.
- **Integration Support**: Works with third-party apps and APIs for extended functionality.

## Installation
### Requirements
- Python 3.10+
- Virtual environment (recommended)
- Dependencies listed in `requirements.txt`

### Setup
1. Clone this repository:
   ```sh
   git clone https://github.com/Lokesh-Sutar/Eric-Personal-Assistant.git
   cd Eric-Personal-Assistant
   ```
2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
3. Run Eric:
   ```sh
   python Eric.py
   ```
